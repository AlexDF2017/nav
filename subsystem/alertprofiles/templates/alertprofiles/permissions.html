Here you can choose which groups are able to recieve which alerts.

Permissions are set by filter groups, so if one wants to deny a certain group to recieve alerts about a certain match field, one must deny the group to recieve alerts from all filter groups that uses a filter that uses that particular match field. In other words: if you have a lot of custom filter groups, that kind of granular control can be hard.

Denying a group to recieve alerts from a certain filter group will not hide the option to add that filter group to one users profile, even though the user won't be able to recieve alerts from it.
